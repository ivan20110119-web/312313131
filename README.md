# 🍕 Telegram-бот доставки пиццы

Бот для заказа пиццы с доставкой на базе **aiogram 3.x**.

## Установка

```bash
cd pizza_bot
pip install -r requirements.txt
```

## Настройка

1. Создайте бота через [@BotFather](https://t.me/BotFather) и получите токен.
2. Узнайте свой Telegram ID через [@userinfobot](https://t.me/userinfobot).
3. Скопируйте `config/local_config.example.py` в `config/local_config.py` и заполните:

```python
BOT_TOKEN = "ваш_токен_от_BotFather"
ADMIN_IDS = [ваш_telegram_id]  # Для уведомлений о новых заказах (можно оставить пустым)
```

Либо задайте переменные окружения:
```bash
set BOT_TOKEN=ваш_токен
set ADMIN_IDS=ваш_id
```

## Запуск

```bash
python bot.py
```

## Функционал

- **🍕 Меню** — 4 вида пиццы (Маргарита, Пепперони, Гавайская, Четыре сыра) с описанием и ценой
- **🛒 Корзина** — добавление нескольких пицц в корзину, оформление заказа
- **📦 Мои заказы** — история заказов пользователя
- **ℹ️ О нас** — информация о пиццерии
- **Админ-панель** — команда `/admin` + пароль `466678001` для просмотра всех заказов

### Процесс заказа (FSM)

1. Выбор пиццы и количества в меню
2. Оформление доставки: улица, дом, квартира
3. Выбор времени: как можно скорее / сегодня / завтра (временные слоты)
4. Комментарий к заказу (опционально)
5. Подтверждение перед отправкой
